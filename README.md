1.软件架构基于 FreeRTOS 实时操作系统搭建，合理划分任务优先级：将传感器数据采集、网络通信处理、云端指令解析、本地执行器控制等模块封装为独立任务，通过信号量与消息队列实现任务间的高效同步与数据交互，确保系统在多任务并发场景下的实时性与稳定性。
2.设备端基于ESP8266接入阿里云物联网平台，通过MQTT协议上报环境数据并接收云端下行指令，配置云平台规则引擎实现设备数据的一发多收。
3.利用阿里云云流转功能将设备消息实时推送至WebSocket服务端，前端页面通过WebSocket长连接接收推送数据，实现设备状态的秒级同步显示与远程控制下发，有效降低了轮询带来的服务端压力。
4.实现了图书阅览区光照自动调节、人体检测、环境温湿度监测等功能，覆盖了从传感器数据采集到云端流转再到网页可视化的完整物联网技术链路。
